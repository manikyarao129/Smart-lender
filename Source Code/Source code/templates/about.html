About Smart Lender

Smart Lender brings together modern web development and machine learning to support loan decisioning in a simple and accountable way.

The platform collects applicant information, evaluates it using a trained predictive model, and returns an approval decision, confidence score, risk level, and next-step recommendation.

What it includes
  • Responsive user experience
  • Flask REST API for predictions
  • SQLite-backed history and dashboard
  • Deployment-ready structure
